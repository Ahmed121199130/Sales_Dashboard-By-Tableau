Sales Dashboard (Tableau)

This project showcases an interactive Sales Dashboard built using Tableau to analyze sales performance and identify key business insights.

Key Features

KPI cards for Total Sales, Profit, Quantity, and Profit Margin

Sales trends analysis by Year & Month

Sales distribution by Region and State

Top 10 products by sales

Sales breakdown by Category & Sub-Category

Quarterly profit analysis

Tools Used

Tableau

Data Visualization & Dashboard Design

Project Objective

To transform raw sales data into clear, actionable insights that support data-driven decision-making.
